## Usage

1. Clone the repository:
   ```
   git clone https://github.com/fernandolin/portfo.git
   cd portfo
   ```
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run the application:
   ```
   python main.py
   ```
4. Open your browser and visit:
   ```
   http://localhost:5000
   ```
